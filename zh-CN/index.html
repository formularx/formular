      Formular

      基于 MobX 的通用表单解决方案 🐺

      开箱即用

      无缝对接且贴合您最常使用的业务组件库（例如 Ant Design）生态。没有魔法，声明式 API 帮助快速上手

      函数响应式编程

      底层由基于 MobX 的 mobx-state-tree 驱动，作为状态容器封装每一个表单项

      高扩展性

      独特的 Markdown 扩展，可嵌入 Demo、可导入外部 Demo 甚至插入自定义 React 组件，使得组件的文档不仅能看，还好用

      如果我们的 demo 是采用 TSX 编写的，在展开源代码后，点击一旁的语言切换按钮，可切换显示成 JSX 的代码 123123

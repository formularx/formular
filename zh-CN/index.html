      Formular

      基于 MobX 的通用表单解决方案 🐺

      原子组件

      抽象化的字段 <Field> 组件，无缝对接且贴合您最常使用的业务组件库（例如 Ant Design）生态。没有魔法，声明式 API 帮助快速上手

      函数响应式编程

      底层由基于 MobX 的 mobx-state-tree 驱动，作为状态容器封装每一个表单项

      高扩展性

      支持快速接入您熟悉的各种组件库
